## 数组 链表 跳表

### 数组 Array
- java , C++  : int a[100];
- python      : list = []
- js          : let arr = [1,2,3]

> 数组可以随机访问任何一个元素,访问第一个元素和访问中间元素,时间复杂度一样 O(1)

缺陷:
- 增加删除插入数组元素  平均O(n)
